
# M3 - Game Launcher

A small game launcher as a project for my school made in python



## Installation

git clone https://github.com/GjinPrelvukaj/M3-Python

```bash
  cd M3-Python
  pip install -r requirements
  python main.py
```
    
## Features

- Friendly GUI
- Easy to read code (Most of the time)
- Fast and bug free
- 4 Games

## Games
- Snake
- Ping Pong - With AI
- Tic-Tac-Toe
- Space Invaders
